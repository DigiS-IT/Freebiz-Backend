# FreeBiz Backend API

> Free & Discounted Service Marketplace - Phase 1 (MVP) Backend

## Tech Stack
- Node.js 18+
- TypeScript 5.4
- Express.js 4.x
- Prisma 5.14
- PostgreSQL 15+

## Getting Started
1. `npm install`
2. `cp .env.example .env`
3. `npx prisma generate`
4. `npm run dev`
